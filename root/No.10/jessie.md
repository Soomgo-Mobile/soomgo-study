## 📚 읽은 내용

- 콘텍스트 규칙
  - 중괄호 `{}` [객체 리터럴, 레이블, 블록, 객체 분해]
  - 레이블 루프? 레이블 블록?
  - else if와 선택적 블록
- 연산자 우선순위
  - && 가 || 보다 우선순위가 높다
- 단락 평가
  - `a && a.cool` 은 boolean 을 뱉지아니하고 a 값을 뱉는다
  - (불필요한 조건이니 처음부터 `a.cool` 로 체크하자)
- 끈끈한 우정
  - && 는 || 보다, ||는 ?보다 각각 우선순위가 높다
- 결합성
  - 좌측에서 우측으로 조건을 결정짓는다? 노우~노우~

## 📚 느낀점

```
[] + {}; // "[object Object]"
{} + []; //0
  ```
